# Rrea Client 2 (C2)

Rrea Client その 2 の業務でなかなか使う機会がない React.js (HOOK)版
Restful API 代わりに、Apollo GraphQL を使っている

バックエンド GitHub：

### スタック

- ⚡️ React.js
- ⚙️ TypeScript
- 🎨 MUI
- 📑 Eslint + Prettier
- 🏭 Vite
- 📦 yarn
- 🔺 GraphQL

### 使用

###### インストール

```bash
yarn install
```

###### 開発

```bash
yarn dev
```

###### コンパイラ

```bash
yarn build
```
