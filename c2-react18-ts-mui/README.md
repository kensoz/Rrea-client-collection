##### Rrea-Client 2 について

メンバー位置と情報の管理システムメンバー用サイト(Client)の React.js 版です。\
業務でなかなか使う機会がない React.js と GraphQL を使って構築しました。



## スタック

- ⚡️ React.js（HOOK）
- ⚙️ TypeScript
- 🎨 MUI
- 📑 Eslint + Prettier
- 🏭 Vite
- 📦 yarn
- 🔺 GraphQL



## 使用

##### インストール

```bash
yarn install
```

##### 開発

[http://localhost:3000](http://localhost:3000) で起動

```bash
yarn dev
```

##### ビルド

```bash
yarn build
```
