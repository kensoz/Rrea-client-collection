## Rrea-Client 2 について

メンバー情報管理システムメンバー用サイト(Client)の React.js 版です。  
React.js と GraphQL を使って構築しました。機能はデフォルト版と同じです。

## メイン機能

+ メンバー位置と情報を表示
+ レスポンシブ対応



## スタック

- ⚡️ React.js（HOOK）
- ⚙️ TypeScript
- 🎨 MUI
- 📑 Eslint + Prettier
- 🏭 Vite
- 📦 yarn
- 🔺 GraphQL



## 使用

##### インストール

```bash
yarn install
```

##### 開発

[http://localhost:3012](http://localhost:3012) で起動

```bash
yarn dev
```

##### ビルド

```bash
yarn build
```
