##### Rrea-Client-Collectionとは

メンバー位置と情報の管理システムの[デフォルトClient](https://github.com/kensoz/Rrea-client)はVue.jsによる構築、公開しましたが、\
React.js、Svelte.jsなどの技術で様々な版も開発しました。
