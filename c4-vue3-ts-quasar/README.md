##### Rrea-Client 4 について

メンバー位置と情報の管理システムメンバー用サイト(Client)のVue.js + Pinia.js版です。\
状態管理ライブラリと業務でよく利用しているQuasarを使って構築しました。

##### メイン機能

+ メンバー位置と情報を表示
+ レスポンシブ対応



## スタック

+ ⚡️ Vue.js 3 + Pinia.js
+ ⚙️ TypeScript
+ 🎨 Quasar
+ 📑 Eslint + Prettier
+ 🏭 Vite
+ 📦 Yarn
+ 🔺 RESTful API



## 使用

##### インストール

```bash
yarn
```

##### 開発

[http://localhost:3014 ](http://localhost:3014) で起動

```bash
yarn dev
```

##### ビルド

```bash
yarn build
```
