# Rrea Client 3 (C4)

Rrea Client その3の Vue.js + Pinia.js版、従来のEventBus方法より、今回は状態管理ライブラリを使った
UIライブラリはよく業務で利用しているQuasarを選定

バックエンドGitHub：

### スタック

+ ⚡️ Vue.js + Pinia.js
+ ⚙️ TypeScript
+ 🎨 Quasar
+ 📑 Eslint + Prettier
+ 🏭 Vite
+ 📦 Yarn
+ 🔺 RESTful API

### 使用

###### インストール

```bash
yarn
```

###### 開発

```bash
yarn run dev
```

###### コンパイラ

```bash
yarn run build
```
